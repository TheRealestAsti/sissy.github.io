HTML alapok – diákbarát összefoglaló

A HTML egy jelölőnyelv, amellyel weboldalakat készítünk. Megmondja a böngészőnek, hogy mi hol és hogyan jelenjen meg.

Egy HTML oldal felépítése

<html>
    <head>
    </head>
    <body>
    </body>
</html>

Minden HTML oldal ebben a sorrendben épül fel.

  • head: az oldal adatait tartalmazza
  • body: az, amit a felhasználó lát

Fontos HTML elemek

HTML elem Mit csinál?
<html> A teljes weboldalt tartalmazza
<head> Az oldal beállításai vannak itt
<title> A böngésző fülén megjelenő szöveg
<meta charset="UTF-8"> A magyar ékezetek helyes megjelenítéséhez kell
<body> Az oldal látható része
<h1> – <h6> Címsorok (a h1 a legnagyobb)
<p> Bekezdés létrehozása
<br> Sortörés
<a href=""> Hivatkozás más oldalra
<img> Kép megjelenítése
<ul>, <ol> Felsorolás vagy számozott lista
<li> Listaelem
<table> Táblázat létrehozása

Összefoglalás

A HTML segítségével készülnek a weboldalak. Nem programozási nyelv, hanem jelölőnyelv. A feladata az oldal szerkezetének megadása.
